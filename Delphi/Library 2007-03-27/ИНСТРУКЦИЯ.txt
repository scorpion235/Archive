Для запуска базы данных (Lib.exe) необходимо, чтобы на компьютере 
был установлен BDE. Для этого достаточно установить Delphi
или C++Builder.

Программа написана на Delphi 6.0, скомпилируется и на 5, и на 7 версии.
В базе данных используется псевдоним (alias) dskExmpl.
Тебе нужно его создать самостоятельно перед компиляцией.
1. Запусти DataBase Desktop.
2. Tools -> Alias Manager.
3. Нажми кнопку "New".
4. Укажи DataBase Alias = dskExmpl и путь к базе данных Path
   (Например: C:\Library\DataBase) - это путь к db-файлам.
5. Нажми "Keep new".

Потом путь к БД можно изменить через SQL Explorer.
