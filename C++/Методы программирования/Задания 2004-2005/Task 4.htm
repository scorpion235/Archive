Задания для практических занятий по теме
«Структуры данных: очередь»

Требуется написать программу, перечисляющую в порядке возрастания все натуральные числа в заданных пределах, которые в разложении на простые множители имеют заданные простые числа.

Для решения этой задачи необходимо использовать алгоритм, разобранный на лекции.

Арифметика больших чисел.

В задание предполагается, что искомые натуральные числа будут достаточно велики, поэтому стандартные типы данных для них не подойдут. Т.о. необходимо реализовать класс для работы со сверхбольшими целыми числами (порядка нескольких тысяч знаков и больше). Базовые операции для реализации данного задания это
1) умножение сверхбольшого числа на заданное число, помещающееся в стандартный тип данных;
2) сравнение двух сверхбольших чисел.

Указание.
Постарайтесь обеспечить возможность расширения Вашего класса сверхбольших чисел, т.к. в одном из последующих заданий снова возникнет необходимость в использовании сверхбольших чисел.

Задание.
1. Реализовать класс сверхбольших чисел.
2. Написать программу, решающую поставленную задачу.
    а) программа должна получать на вход два числа - границы отрезка натурального ряда, в котором будет осуществляться поиск, а также список простых чисел из промежутка от 2 до 108.
    б) выходными данными служат список найденных чисел и время работы программы.
3. Разработать тесты для Вашей программы.

Задание рассчитано на две недели. Срок сдачи 18-19 ноября.
Напоминаю, что каждое практическое занятие, на котором студент не отчитался за проделанную работу рассматривается как -0.5 задания.
