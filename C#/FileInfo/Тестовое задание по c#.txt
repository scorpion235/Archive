Написать небольшую программу, которая должна выполнять следующие функции:

- посчитать объём указанного файла/директории (рекурсивно)

- давать пользователю возможность выбора единиц измерения (гигабайт, мегабайт, килобайт, байт)

- давать пользователю возможность задать расширение файла, и искать суммарный размер только тех файлов, расширение которых совпадают с заданным.

 

 

Оцениваться будут следующие параметры кода:

- структура программы

- используемые библиотеки/алгоритмы/системные функции

- следование соглашениям по написанию исходного кода C# (C# Coding Conventions, наименование переменных, функций, классов и т.д.)

- объём кода

- отсутствие ошибок при компиляции и запуске

- удобство пользования

 

Плюсами будет считаться:

- обработка ошибок

- выводом статуса выполнения и прогресса

- производительность.

 

Программа должна запускаться на компьютере с установленным .NET (версии не ниже 4)

Язык - C#

Среда разработки - Visual Studio (не ниже 2010)
